Bob 16 is an js based engine making interactions with canvas api a lot easier its an game enigne on level of pygame ,still in early early access 
also has its custom editor bob 16 eidtor accessible at maksydab.github.io/bob16/
the custom editor interacts with bob16 api  lua endpoint to compile and execute an lua based all in one editor for this enigne supporting making art with limited pallete making tilemaps for levels and terminal

bob16 editor has an native save and load format that uses png files as in single png file to store entire game data to load later
