Bob 16 Manual

Bob 16 is a demo mini-game engine that saves its games into PNG files.It was made form scratch using canvas api in js by single dev , maksydab

it's an work in progress early version,you may encounter many many bugs

To use it:

  • Open the T tab (Terminal)
  • Run the command urn to start the game
  • Use save to save your game and load to load a game

üîß Lua Engine Methods
clear(pidx)
  - Clears the pixel buffer at with color picked from a pallete by index (default 0)

drawPixel(x, y, idx)
  - Draws pixel at (x, y) with color index idx (default 7)

drawRect(x, y, w, h, idx)
  - Draws filled rectangle starting at (x, y), width w, height h, color idx (default 7)

drawLine(x1, y1, x2, y2, idx)
  - Draws line from (x1, y1) to (x2, y2), color idx (default 7)

drawImage(index, x, y, scale)
  - Draws sprite from atlas at index, positioned at (x, y), scaled by scale (default 1)

drawTilemap(scale=1, ox=0, oy=0)
  - Draws the tilemap scaled by scale, offset by (ox, oy)

getTile(x, y)
  - Returns a table {x=..., y=..., id=...} for tile at (x, y), or nil

setTile(x, y, id)
  - Sets tile at (x, y) to tile id

isKeyDown(key)
  - Returns true if the given key (string) is currently pressed

The engine outputs the saved game as a PNG image that you can freely edit. Just beware that some editors might overwrite metadata.
